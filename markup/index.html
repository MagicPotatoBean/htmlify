The things I have made


I have many projects, most available on my github account, some much better than others, but here are some of the ones I am most proud of:

  • HTML-ify, the tool I made so that I could have consistent formatting on all of my pages by converting markdown into stylised HTML (I wrote this page in pure markdown and used htmlify to turn it into html!)
  • Host, a rust-script program which can take a list of files and a list of addresses, and will run a basic HTTP server to share these files for you.
  • Scriptify, another rust-script program which takes a cargo workspace and converts it into a single rust-script file (including dependancies and multi-file programs).
